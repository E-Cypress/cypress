## Para rodar:
- Clone o repositorio
- Instale as dependencias pelo terminal com o comando:

```cmd
npm i
```

- Abra dois terminais e rode os seguintes comandos:


_(esse roda o site localmente em http://localhost:3000/)_
```cmd
npm start
```

_(esse roda a interface do cypress)_
```cmd
npx cypress open
```
